# Frontend - Mini Social

This is a minimal React frontend built with Vite.

Install

1. cd frontend
2. npm install

Run (development)

npm run dev

Build

npm run build

Configuration

Set environment variable VITE_API_URL to point to backend API (e.g. http://localhost:4000) if the backend is hosted elsewhere.

Deployment

Build the frontend (`npm run build`) and deploy the `dist/` folder to Vercel / Netlify.
